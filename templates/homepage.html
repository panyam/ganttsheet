Usage

Installation

Gantt Sheet is available on the G Suite Marketplace. Select Install to begin using it in Google Docs.

Starting the add-on

Open a new sheet and Click on Add-Ons >> Gantt Sheet >> Start.

Resetting the sheet

To clear removing project tracking and Gantt Charts from the current sheet click on Add-Ons >> Gantt Sheet >> Reset.

Limitations

  • Maximum calendar view of of one year.
  • Project Tracker template not yet customizable.

Contributing

Setup

            
            $ cd gdocs
            $ npm install

Build

            
            $ cd gdocs
            $ npm run build

Pushing Changes

Command Description
npm run push:server All GAS Server Scripts
npm run push:client just main.html
